# 📸 图片收集站

一个简单易用的本地图片收集工具，支持拖拽上传和手机访问。

## ✨ 功能特点

- 🖱️ 拖拽上传图片（电脑端）
- 📱 手机拍照/相册上传
- 🖼️ 自动按日期分组
- 📋 一键复制到剪贴板
- 🗑️ 快速删除图片
- 🌓 自动适配深色模式
- 📱 响应式设计，手机电脑都好用

## 🚀 快速开始

### 1. 安装依赖

```bash
cd ~/image-collector
npm install
```

### 2. 启动服务

```bash
npm start
```

### 3. 访问应用

**电脑访问：**
```
http://localhost:3000
```

**手机访问：**
```
http://192.168.x.x:3000  （终端会显示具体地址）
```

手机扫描终端显示的二维码即可访问

## 📝 使用说明

### 电脑端
1. 直接拖拽图片到上传区域
2. 或点击"选择文件"按钮
3. 点击图片查看大图
4. 点击"复制"按钮，即可粘贴到文档

### 手机端
1. 连接与电脑相同的 WiFi
2. 浏览器打开服务地址或扫码
3. 点击上传区域选择照片或拍照
4. 上传后电脑端立即可见

## 💡 提示

- 图片自动按日期归类存储在 `uploads` 文件夹
- 支持的格式：JPG、PNG、GIF、WebP、BMP
- 手机和电脑需连接同一 WiFi 网络
- 电脑需保持运行状态

## 🔧 可选配置

### 修改端口
编辑 `server.js` 第 8 行：
```javascript
const PORT = 3000;  // 改为其他端口
```

### 设置开机自启动
可以使用 PM2 等工具管理进程，或添加到系统启动项。

## ❓ 常见问题

**Q: 手机无法访问？**
- 确保手机和电脑在同一 WiFi
- 检查电脑防火墙设置

**Q: 上传失败？**
- 确保文件是图片格式
- 检查磁盘空间是否充足

**Q: 复制功能不work？**
- 需要使用 HTTPS 或 localhost
- 部分浏览器可能不支持
